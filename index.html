Sobre

Sou estudante de Licenciatura em Computação pelo Instituto Federal do Piauí - Campus Zona Sul.

Esta página está sendo desenvolvida como parte da disciplina Introdução à Programação Web, orientada pelo professor Msc. Jefferson de Sousa Silva. O objetivo deste projeto é aplicar os seguintes conceitos:

  • Estruturar documentos web utilizando HTML;
  • Adotar as melhores práticas recomendadas pelo W3C (Web Standards);
  • Estilizar e organizar visualmente as páginas usando CSS;
  • Programar funcionalidades interativas com JavaScript;
  • Desenvolver sites responsivos, adaptados a diferentes dispositivos.

Atividades

Atividade 1

Atividade voltada para a utilização da Tag anchor para a referenciação de recurso internos ou externos. E também a organização do repositório em pastas.
Atividade 2

Atividade voltada para a utilização de mais Tags HTML e atributos de Tags para o destaque de textos, divisões de seções, listagem itens, numeração de itens e imagem.
Atividade 3

Atividade voltada para a utilização de tabelas por meio da tag table, evidenciação do cabeçalho da tabela, mesclar linhas e colunas e ajuste de bordas, linhas, colunas e células.
Atividade 4

Atividade voltada para a utilização da Tag form para a criação de formulários que contém elementos que permitem ao usuário entrar com informações, como: campos de texto, menus drop-down, botões radiais, caixas de seleção.
Atividade 5

Atividade voltada para a utilização de CSS (Cascading Style Sheets) para adicionar estilos (cor, fontes, espaçamento, etc.) a um documento web.
Atividade 6

Atividade voltada para a utilização das tags div e span na criação de divisões lógicas e estruturais de páginas, e conteiner inline para conteúdos fraseados.
Atividade 7

Atividade voltada para a utilização de Família de Fonte, Propriedade Font-size e Propriedade Font-style.
Atividade 8

Atividade voltada para a utilização de Box Model, ou seja, tags estilizadas que se comportam como caixas.
Atividade 9

Atividade voltada para a utilização de propriedades de posicionamento do CSS que tem como utilidade ampliar as possibilidades de um layout.
Atividade 10

Atividade voltada para a utilização das novas tags (nav, header, footer) para maior eficiência e lógica para estruturar um site em HTML5.
Atividade 11

Atividade voltada para a utilização das novas tags (article, section e aside) para maior eficiência e lógica para estruturar um site em HTML5.
Atividade 12

Atividade voltada para a utilização do atributo Data para costumização de dados.
Atividade 13

Atividade voltada para a utilização do normalize CSS, uso do CSS3, Propriedade border-radius e propriedade box-sizing.
Atividade 14

Atividade voltada para a utilização de sombras, box-shadow e gradiente.
Atividade 15

Atividade voltada para a utilização de transição e animação usando CSS.
Atividade 16

Atividade voltada para a utilização de Design Responsivo, Media Types, CSS Media Queries e Bootstrap.
Atividade 17

Atividade voltada para a utilização do Sistema Grid.
Atividade 18

Atividade voltada para a construção de Formulários e Tabelas usando o Bootstrap.
Atividade 19

Atividade voltada para a utilização de Botões e Imagens Responsivas usando o Bootstrap.
Atividade 20

Atividade voltada para a utilização de NavBar e Janelas modais usando o Bootstrap.
Atividade 21

Atividade voltada para a utilização de alert, tooltip e popover usand o Bootstrap.
Atividade 22

Atividade voltada para a utilização de JavaScript no desenvolvimento de páginas de websites.
Atividade 23
