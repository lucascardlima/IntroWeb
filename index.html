Esta página está sendo desenvolvida como proposta da disciplina Introdução a Programação Web ministrada pelo Docente Msc. Jefferson de Sousa Silva
As atividade abaixo tem como objetivo:
- Estruturar documentos web usando as linguagens HTML;
- Utilizar as melhores práticas – web standards – relacionadas pelo W3C;
- Formatar a apresentação de documentos web utilizando CSS;
- Programar recursos da página utilizando javascript;
- Criar Sites Responsivos
